= GPU Resource Management on OpenShift

A Red Hat Showroom lab guide for GPU resource management on OpenShift, covering fake GPU simulation, GPU booking, hardware profiles, Kueue integration, and workload preemption.

== Quick Start

. Clone this repo and preview locally:
+
[source,sh]
----
podman run --rm --name antora -v $PWD:/antora -p 8080:8080 -i -t ghcr.io/juliaaano/antora-viewer
----
+
On SELinux systems, append `:z` to the volume mount.

. Edit content in `content/modules/ROOT/pages/`
